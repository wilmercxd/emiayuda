# EMI FALCK - Asistente de Ventas v3.0

Herramienta integral para agentes de call center de EMI Colombia.

## Características

- ✅ 8 planes con tarifas 2026
- ✅ Beneficios dinámicos por plan
- ✅ 14 ciudades con números locales
- ✅ Extractor ADRES
- ✅ Matriz de objeciones
- ✅ Descarga de resumen TXT
- ✅ Interface responsive

## Deploy

Simplemente abre `index.html` en cualquier navegador. Funciona sin servidor.
